Chatbot de Recomendações com Gemini

Inteligência Artificial no Google Colab

Desenvolvi um chatbot interativo no Google Colab utilizando o modelo de linguagem Gemini do Google AI Studio. O chatbot oferece recomendações personalizadas de produtos ou entretenimento com base nas preferências do usuário. Veja o código no GitHub.

Desafios:

  • Criação de interface amigável no Colab.
  • Comunicação entre frontend e backend (JavaScript e Python).
  • Gerenciamento de estados do Dash.
  • Extração e formatação de dados da resposta do Gemini.

Tecnologias:

  • HTML
  • CSS
  • JavaScript
  • Python
  • Google Colab
  • API do Google AI Studio
  • Modelo Gemini
  • IPython.display.HTML
  • google.colab.kernel.invokeFunction
  • Expressões Regulares

Resultados: Chatbot funcional que permite ao usuário receber recomendações personalizadas de acordo com suas preferências.
